# MememoriLauncher
 APK版メメントモリを自動でアップデートするアプリ

## インストール方法
1. リリースタブからapkファイルをダウンロードする  
https://github.com/EldoDebug/MememoriLauncher/releases
2. Android端末に入っているファイルマネージャーでapkファイルを起動し、インストール (公式apk手順と同様)
